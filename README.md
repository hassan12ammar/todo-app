# TODO app
This is a simple todo app built with React and NestJS. 

## Features

### Auth
-   signin
-   signup
-   signout

### Todo
-   Get user's todos
-   Create a todo
-   Update a todo
-   Delete a todo

## Tech Stack

### Backend
-   TypeScript
-   NestJS
-   Prisma
-   SQLite
-   JWT (with Bearer strategy)

### Frontend
-   React
-   TypeScript
-   React Query
-   React Router
-   Axios

## Run the app

### Backend

```bash
cd server
pnpm install
pnpm start:dev
```

### Frontend

```bash
cd client
pnpm install
pnpm dev
```
